```

├── index.html          # الصفحة الرئيسية
├── merchant.html       # أنواع التجار (SME / Enterprise)

├── tracking.html       # تتبع الشحنة
├── integrations.html   # التكاملات مع المنصات
├── storage.html        # خدمة التخزين
├── demo.html           # ديمو تفاعلي للتاجر
├── about.html          # من نحن والفريق
├── contact.html        # تواصل معنا
├── app.html            # تطبيق جمبك
├── blog.html           # المدونة

├── register.html       # إنشاء حساب
├── dashboard.html      # لوحة تحكم التاجر
├── base.css            # الأنماط المشتركة

├── hero-map.png        # صورة الهيرو
└── warehouse.png       # صورة المستودع
```

## ⚙️ التقنيات

- **HTML5 + CSS3** — بدون أي framework
- **Vanilla JS** — تفاعلية بسيطة وسريعة
- **Google Fonts** — خط Cairo
- **Calendly** — حجز اجتماعات المبيعات
- **GitHub Pages** — استضافة مجانية

---

## 📅 Calendly

زر **"تواصل مع المبيعات"** في الصفحات التالية يفتح popup لحجز اجتماع:

- `merchant.html` — قسم الأعمال الكبيرة
- `pricing.html` — الباقة Enterprise
- `storage.html` — باقة Enterprise للتخزين

الرابط: `https://calendly.com/anas-msd-ramsees/30min`

## 📱 دعم الموبايل

- **Hamburger menu** تلقائي عبر `nav.js` في كل الصفحات
- تصميم responsive كامل من 480px حتى 1440px+
- RTL-safe على جميع الأجهزة
